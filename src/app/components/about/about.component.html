LocMer

LocMer es un chat online web, sin la necesidad de registrarse o poner un número celular para poder intercambiar mensajes entre personas con la confiabilidad de que los mensajes enviados y recibidos no se guardan en alguna base de datos o servidor haciendo que los mensajes enviados y recibidos solo permanecan en el navegador de las personas que están chateando.

Como funciona

Este nombre se mostrara a los demás usuarios a la hora de intercambiar mensajes

A la hora de crear el chat te pedirá el nombre que dees darle al chat a crear

Se crea el chat con el nombre, genera un código que es el que tendrpa que pasar a las personas que quieren que estpen en su chat

Al añadir un chat se te pide el nombre que le quiera dar al chat y el código que le pasaron de un chat ya previamente creado por alguien
